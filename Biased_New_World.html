Biased New World

In the digital age, our media consumption patterns have become increasingly influenced by powerful recommendation algorithms. While these algorithms aim to enhance user experience by suggesting personalized content, they inadvertently contribute to the perpetuation of often unknown individual biases. As users engage with content tailored to their preferences, we are inadvertently shielded from diverse perspectives and alternative viewpoints.

This project aims to raise the user’s awareness of their media consumption habits and recommend different content than the one suggested by the recommendation algorithms of the media platforms used.

To do this the user uploads their activity history from Netflix and/or Spotify which can be downloaded from the app. Access to this data is protected in the European Union under the GDPR (General Data Protection Regulation) adopted in 2016.

We developed a GUI (Graphical User Interface) to make the project more accessible to the end user. The GUI was developed using the Python library Tkinter.

We display the user’s watching/listening history throughout time as well as a world map coloured based on what percentage of content was shot and produced in that country. We interpret these results as a proxy for homogeneity/heterogeneity of world views expressed through the works of art. We also included a recommendation algorithm where the user, using the bar slides available, can quantify the distance desired between the recommendation to be provided and the content already consumed. Therefore, setting the bar slide to a very low value ensures a recommendation closer to the one expected using the app’s recommendation system and vice-versa.



This project was developed by Afonso Ribeiro, Nuno Machado and Tiago Miranda for the Innovation and Development Laboratory course at Instituto Superior Técnico in Lisbon, Portugal.
